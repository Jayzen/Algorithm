#数组vs列表

```ruby
数组：数组中每个元素包括该元素的值和下表。
列表：列表中的每个元素包括该元素的值和下一个元素的地址。
```
#数组和列表的大O表示法的运行时间

|          |      数组      |  链表 |
|----------|:-------------:|------:|
|  读取    |  O(1)         |O(n)    |
| 插入     |  O(n)         |O(1)   |
| 删除     | O(n)          |O(1)   |
备注：链表中的删除为O(1) 其实是O(n)+O(1)，所有的数组删除首先要进行数据查询
